01 — About

Built for the long game.

I've spent years owning and managing Minecraft networks — from the first server config to a fully-fledged community with its own economy, staff team, and identity. Along the way I've learned what actually keeps a network alive: stable performance, fair monetization, and a community that feels heard.

I've run three networks across different eras of my journey, each one teaching me more about server architecture, plugin development, and what players genuinely want out of a server. I'm now channeling everything I've learned into ReefMC — my most ambitious project yet.

Alongside operations, I'm developing my technical side — writing YAML configs daily, learning Java to build my own plugins, and using HTML to ship the tools and pages my networks need.
